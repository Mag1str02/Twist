# Отладка

## Атомики

Для отладочного логирования используйте:

- Метод `DebugLoad` у `atomic<T>`
- Метод `DebugTest` у `atomic_flag`

В `build::Thr()` для них отключен fault injection, в `build::Sim()` – прерывание, 
они не влияют на дайджест симуляции.
